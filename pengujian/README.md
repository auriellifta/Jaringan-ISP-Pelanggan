# Pengujian Jaringan

Folder ini digunakan pada Progres 3 untuk menyimpan bukti pengujian.

Berkas yang perlu ditambahkan:

- `ping_test.png`: hasil ping client ke server lokal dan server uji.
- `traceroute.png`: jalur trafik dari LAN pelanggan menuju server uji.
- `routing_table.txt`: hasil pemeriksaan tabel routing setiap router.
- `nat_test.png`: bukti translasi NAT/PAT pada R-CPE pelanggan.
