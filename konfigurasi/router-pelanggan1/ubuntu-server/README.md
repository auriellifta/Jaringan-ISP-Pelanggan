# Ubuntu Server Lokal

Folder ini digunakan untuk dokumentasi Ubuntu Server Lokal pada LAN pelanggan.

Rencana alamat:

- IP address: `192.168.10.20/24`
- Default gateway: `192.168.10.1`

Pada Progres 2 atau Progres 3, tambahkan catatan konfigurasi dan screenshot pengujian layanan lokal apabila diperlukan.
