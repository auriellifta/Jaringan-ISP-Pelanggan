# Konfigurasi Router ISP

Folder ini digunakan pada Progres 2 dan Progres 3 untuk menyimpan konfigurasi router ISP.

Berkas yang perlu ditambahkan setelah implementasi:

- `config_isp.txt`: konfigurasi final router ISP.
- `screenshot_isp.png`: tangkapan layar konfigurasi atau hasil verifikasi router ISP.
